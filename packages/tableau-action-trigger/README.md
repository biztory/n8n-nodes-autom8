# @biztory/n8n-nodes-autom8-tableau-action-trigger

n8n community node that receives data from the **Autom8** Tableau dashboard extension via webhook and surfaces it as workflow input. Pair it with the Autom8 Extension on a Tableau dashboard to trigger n8n workflows from user actions on the viz.

ℹ️ Full documentation: https://biztory.atlassian.net/wiki/spaces/A8/pages/1123254273/n8n+Node

## Installation

Follow the [installation guide](https://docs.n8n.io/integrations/community-nodes/installation/) in the n8n community nodes documentation, and search for `@biztory/n8n-nodes-autom8-tableau-action-trigger`.

## Usage

1. Add the **Autom8 Tableau - Data Action Trigger** node to a new workflow.
2. Pick a unique value for **Webhook Path** (e.g. `autom8-customer-satisfaction`).
3. Copy the resulting webhook URL into the Autom8 Extension on your Tableau dashboard.

For a step-by-step introduction, see the [Getting Started page](https://biztory.atlassian.net/wiki/spaces/A8/pages/1123090433/Getting+Started).

## Compatibility

Tested and supported starting with n8n version **2.13.3**.
